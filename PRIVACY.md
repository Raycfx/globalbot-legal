# Politique de confidentialité – Ray BOT

Global BOT respecte la confidentialité des utilisateurs et collecte uniquement les données nécessaires à son fonctionnement.

## Données collectées
Global BOT peut collecter et stocker :
- Identifiants Discord (ID utilisateur, ID serveur, ID salon)
- Données liées à la modération (avertissements, mutes, bans, logs)
- Paramètres de configuration du serveur

## Utilisation des données
Les données collectées sont utilisées uniquement pour :
- Assurer les fonctionnalités de modération et d’administration
- Permettre le bon fonctionnement des commandes et automatisations
- Améliorer la stabilité et les performances du bot

## Partage des données
- Aucune donnée n’est vendue, échangée ou partagée avec des tiers.
- Les données ne sont accessibles qu’aux administrateurs du bot.

## Conservation des données
Les données sont conservées tant que le bot est présent sur le serveur.
Elles peuvent être supprimées sur demande via le serveur Discord officiel.

## Sécurité
Des mesures raisonnables sont mises en place pour protéger les données stockées.

## Modifications
Cette politique de confidentialité peut être modifiée à tout moment.

Dernière mise à jour : Décembre 2025
